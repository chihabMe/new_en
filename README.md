## Tech Stack

- **UI Components**: Radix UI
- **Animations**: Motion
- **Forms**: React Hook Form with Zod validation

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Set up your environment variables
4. Run the development server: `pnpm dev`
